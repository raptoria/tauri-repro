Tauri Test repo

Apps built with Tauri, Vite, React & Tailwind

## Getting Started

Make sure you have the Tauri prerequisites installed:
https://tauri.app/start/prerequisites/

## Install dependencies

```sh
pnpm i
```

## Build

```sh
pnpm build
```

## Development commands

```
- pnpm ios
- pnpm web
- pnpm android
- pnpm desktop
```
